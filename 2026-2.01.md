学习git命令
git config --global user.name "你的名字"，更改用户名
git clone [链接]克隆一份，仓库上的复制一份到本地
git pull是终端打开文件位置之后用的，需要先进行cd，更新到话是更新我打开的这个文件夹
git status 查看文件修改状态
更新代码
第一步：git add . 更新到暂缓区，【.】 代表把当前目录下所有修改都放进去；如果是 git add 文件名  则是只放这一个文件。
第二步：git commit -m "你的备注信息" 提交更新和备注内容
第三步：git push 更新到github

终端命令
pwd 查看文件路径
ls 查看文件夹内有哪些文件 ls -a可以查看隐藏文件
cd [路径]相当于用鼠标双击进文件夹，或者按“返回”键
cd code：进入名为 code 的文件夹（相当于双击进入）。
cd ..：退回到上一级（相当于点“返回”箭头）。注意 cd 和 .. 中间必须有空格！
cd ~：一键回到你的“主目录”（用户主目录）。
mkdir [文件夹名] 新建文件夹
touch [文件名] 新建文件
open .打开当前文件夹窗口mac电脑
start .打开当前文件夹窗口window电脑
git init 把这个文件夹变成 Git 仓库
